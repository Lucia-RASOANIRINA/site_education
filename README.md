Déscription
Ultimate Child Education Center est un site one-page moderne, entièrement responsive et coloré que j’ai recréé en version miroir. Il présente un centre éducatif pour enfants de 3 à 12 ans avec un design chaleureux, ludique et professionnel. Le site inclut une navigation fluide, des animations subtiles, un formulaire de contact interactif avec des popups personnalisés (succès et erreur), des témoignages, les programmes éducatifs, les événements et une section actualités. Une attention particulière a été portée à l’expérience utilisateur et aux détails visuels enfantins.
Ce projet démontre ma capacité à reproduire fidèlement des sites éducatifs modernes, attractifs et orientés conversion avec un code propre et soigné.

Technologies utilisées (Tech Stack) :
HTML5
CSS3 (Flexbox, Grid, animations personnalisées, responsive design)
Vanilla JavaScript (navigation active au scroll, gestion du formulaire, popups personnalisés)
Font Awesome 6 (icônes)
Remixicon (icônes complémentaires)
Google Fonts (Inter)

Autres caractéristiques techniques :
Design 100% responsive (mobile-first)
Aucune framework (pure vanilla)
Animations CSS + JavaScript
Form validation avec messages d’erreur/succès thématiques
